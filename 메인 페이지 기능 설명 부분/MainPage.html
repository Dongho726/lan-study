🔍 기능 설명 🔍

📌Live
실시간 스트리밍과 채팅 기능으로 공부에 활력을 더해봐요!
▷실시간 스트리밍 기능
-Watch란에 사용자 닉네임 입력 후 '->' 버튼 클릭 시 해당 사용자의 실시간 스트리밍 화면을 볼 수 있습니다.
-Host 옆 '->' 버튼 클릭 시 직접 호스트가 되어 실시간 공부 스트리밍을 진행할 수 있습니다.
▷실시간 채팅 기능
-특정 사용자의 실시간 스트리밍을 시청하며 공부에 동기 부여가 되는 말 등을 실시간으로 입력할 수 있습니다.

📌To Do List
오늘의 할 일을 작성하고 하나씩 체크해가는 기쁨을 누려봐요!
▷입력란에 Task 작성 후 Enter키를 누르면 To Do List에 할 일이 추가됩니다.
▷✅를 누를 경우 Task가 Finished로 넘어갑니다.
▷❌를 누를 경우 입력한 Task가 삭제됩니다.
▷⏪를 누를 경우 Finished로 넘어갔던 Task가 다시 할 일로 추가됩니다.

📌D-Day
중요한 일정이나 시험을 디데이 기능으로 기록해 잊지 말아요!
▷d-day명과 year, month, day 입력 시 해당 일정의 디데이가 추가됩니다.
▷❌를 누를 경우 입력한 일정이 삭제됩니다.
▷입력 예시: year란에 '2020', month란에 '09', day란에 '1'
🚨주의🚨: month란에 한 자리수 달 입력 시 '08'과 같이 작성해주시길 바랍니다.
(month란에 '8'과 같이 입력 시 NaN 오류가 발생합니다.)

📌White Noise
백색소음, 학습 관련 뇌파소리와 함께라면 집중도 200% 상승!
▷공부하며 듣고 싶은 백색소음 혹은 뇌파소리 글씨나 이미지를 클릭하면 새로운 페이지에 영상 창이 열립니다.
▷출처:공부의 신 강성태 / 잠박사 Jambaksa
